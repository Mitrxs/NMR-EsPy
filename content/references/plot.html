nmrespy.plot¶

Module for plotting estimation results.

class nmrespy.plot.ResultPlotter(data, result, expinfo, *, plot_residual=True, plot_model=False, residual_shift=None, model_shift=None, shifts_unit='ppm', region=None, data_color='#000000', residual_color='#808080', model_color='#808080', oscillator_colors=None, show_labels=True, stylesheet=None)[source]¶

Class for generating figures of estimation results.

Parameters

  • data (ndarray) – Time-domain data.

  • result (ndarray) –

    Parameter estimate, of form:

    • 1-dimensional data:

      parameters = numpy.array([
   [a_1, φ_1, f_1, η_1],
   [a_2, φ_2, f_2, η_2],
   ...,
   [a_m, φ_m, f_m, η_m],
])

    • 2-dimensional data:

      parameters = numpy.array([
   [a_1, φ_1, f1_1, f2_1, η1_1, η2_1],
   [a_2, φ_2, f1_2, f2_2, η1_2, η2_2],
   ...,
   [a_m, φ_m, f1_m, f2_m, η1_m, η2_m],
])

  • expinfo (ExpInfo) – Experiment information.

  • shifts_unit (str) – Units to display chemical shifts in. Must be either 'ppm' or 'hz'.

  • region (Optional[Iterable[Tuple[float, float]]]) – Boundaries specifying the region to show. N.B. the units region is given in should match shifts_unit .

  • plot_model (bool) – If True, plot the model generated using result. This model is a summation of all oscillator present in the result.

  • plot_residual (bool) – If True, plot the difference between the data and the model generated using result.

  • model_shift (Optional[float]) – Specifies a translation of the residual plot along the y-axis. If None, a default shift will be applied.

  • residual_shift (Optional[float]) – Specifies a translation of the residual plot along the y-axis. If None, a default shift will be applied.

  • data_color (Any) – The colour used to plot the data. Any value that is recognised by matplotlib as a color is permitted. See here for a full description of valid values.

  • residual_color (Any) – The colour used to plot the residual. See data_color for a description of valid colors.

  • model_color (Any) – The colour used to plot the model. See data_color for a description of valid colors.

  • oscillator_colors (Any) –

    Describes how to color individual oscillators. The following is a complete list of options:

    • If a valid matplotlib color is given, all oscillators will be given this color.

    • If a string corresponding to a matplotlib colormap is given, the oscillators will be consecutively shaded by linear increments of this colormap. For all valid colormaps, see here

    • If an iterable object containing valid matplotlib colors is given, these colors will be cycled. For example, if oscillator_colors = ['r', 'g', 'b']:

      • Oscillators 1, 4, 7, … would be red (#FF0000)

      • Oscillators 2, 5, 8, … would be green (#008000)

      • Oscillators 3, 6, 9, … would be blue (#0000FF)

    • If None, the default colouring method will be applied, which involves cycling through the following colors:

      • #1063E0

      • #EB9310

      • #2BB539

      • #D4200C

  • show_labels (bool) – If True, each oscillator will be given a numerical label in the plot, if False, the labels will be hidden.

  • stylesheet (Optional[str]) – The name of/path to a matplotlib stylesheet for further customaisation of the plot. See here for more information on stylesheets.

Return type

None

displace_labels(labels, displacement)[source]¶

Displace labels.

Often the default location of the labels can be undesirable. This method aims to make it simple to re-position the labels.

Parameters

  • labels (Iterable[int]) – The identities of the labels to displace.

  • displacement (Tuple[float, float]) – The amount to displace the labels relative to their current positions in the x- and y-directions, respectively. The displacement uses the axes co-ordinate system. Both values provided should be less than 1.0.

save(path, fmt='png', dpi=400, force_overwrite=False, fprint=True)[source]¶

Save the result figure.

Parameters

  • path (Union[str, Path]) – Path to save the figure to. Suffix does not need to be included.

  • fmt (str) – File format. Must be one of "svgz", "ps", "emf", "rgba", "raw", "pdf", "svg", "eps", "png"

  • dpi (int) – Dots per inch.

  • force_overwrite (bool) – If False, and the path provided to save the file to already exists, the user is prompted to indicate whether they are happy to overwrite. If True, the file will always be overwritten.

  • fprint (bool) – Whether or not to output information to the terminal.
